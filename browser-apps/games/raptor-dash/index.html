Raptor Dash

Jump, dodge, and ride the thermals. Procedurally generated levels - every run is new.

Controls:

Space / Up / Click: Jump
Down Arrow: Crouch
Double-tap Jump: Boost (if available)
Start Button: Begin/Restart game
Pause Button: Pause/Resume
Score: 0
Level: 1
