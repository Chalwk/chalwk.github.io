Asteroids Adaptation

Classic arcade game with spaceship controls and asteroid destruction.

Controls:

Arrow Left: Rotate left
Arrow Right: Rotate right
Arrow Up: Thrust forward
Space: Shoot
R: Restart game
Score: 0
Lives: 3
