Ultimate Tic-Tac-Toe p5.js

Win 3 small boards in a row to win the game. Next move is dictated by previous move's position.

Controls:

Click: Place your mark
Spacebar: Reset game
M key: Toggle game mode
AI Mode: O plays automatically
Current Player: X
Next Board: Any
Status: Player X's turn
X: 0 Ties: 0 O: 0

Shortcuts: Space = Reset | M = Toggle Mode | 1-9 = Quick Board Select
