Ultimate Tic-Tac-Toe

Win 3 small boards in a row to win the game. Next move is dictated by previous move's position.

Rules:

Player X: Starts the game
Player O: Second player
Next Board: Determined by opponent's last move position
Win Condition: Win 3 small boards in a row
Reset: Start a new game
Current Player: X
Next Board: Any
Status: Player X's turn
