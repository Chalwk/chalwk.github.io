Password Security Guidance

Best practices and recommendations for secure passwords.

  • NIST recommendations: Use passwords with 12+ characters, avoid reused passwords, and prefer long passphrases for memorability.
  • zxcvbn strength scoring: Scores range from 0 (very weak) to 4 (very strong). Aim for scores 3-4 for important accounts.
  • Character diversity: Use multiple character sets (upper, lower, numbers, symbols) to increase complexity.
  • Password length: Longer passwords are generally more secure than complex short ones. Consider using passphrases.
  • Uniqueness: Never reuse passwords across different websites or services.
  • Password managers: Use a reputable password manager to store unique passwords for all your accounts.
  • Two-factor authentication: Enable 2FA wherever possible for added security beyond passwords.
