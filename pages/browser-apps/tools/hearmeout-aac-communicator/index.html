    How to Use

    ×
    • Tap a symbol - adds it to your phrase bar.
    • Long-press a symbol (≈ 0.5s) - speaks the symbol text without adding it.
    • Phrase bar - drag items to reorder; tap an item to remove it.
    • Speak - reads the whole phrase aloud.
    • Clear - empties the phrase bar.
    • Left / Right arrows - switch between category pages.

    Settings (⚙️ gear icon)

    • Add Symbol / Edit Mode - build your own board.
    • Save - commit changes to this device.
    • Export / Import - back up or share symbols & settings.
    • Manage Categories - rename or delete categories (also merges categories from symbols).
    • Additional Settings - theme (light/dark/high-contrast), speech volume, voice, and board grid size (auto-responsive or fixed 3×4/4×6/6×8).

    Keyboard shortcuts (when the board is focused)

    • E - toggle edit mode.
    • Ctrl+Z - undo last phrase change.
    • Enter or Space - add focused symbol.
    • Escape - close settings dropdown.
    • ← / → - previous / next category
    • 1-9 - speak card 1-9; 0 speaks card 10
    • Ctrl+1-Ctrl+9 - speak cards 11-19; Ctrl+0 speaks card 20
    • Alt+1-Alt+9 - speak cards 21-29; Alt+0 speaks card 30
    • Ctrl+Alt+1-Ctrl+Alt+9 - speak cards 31-39; Ctrl+Alt+0 speaks card 40
